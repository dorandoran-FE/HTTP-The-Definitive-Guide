## 💫 진행방식

- 해당 주차의 책 내용을 읽고 모르는 부분 조사

- 깃허브 각 장 폴더 하위에 본인 이름의 markdown 파일 업로드

  ```
  본인 [이름.md]
  ```

  파일에 공부한 내용 정리해 올리기

  - 공부한 내용 정리
  - 인상 깊은 점
  - 새로 알게된 개념

- Commit Convention : 본인 이름 : 특정장-제목

  ``` 
  채리: 1장-네트워크 기본
  ```

- 스터디 당일에 작성한 내용에 대한 발표 + 주요 내용에 대한 토의
